ReWindows

学期结束了，把windows恢复出厂重新倒腾一遍。不得不说ai真是太好用了，放在以前不知道遇到的问题要google多久，现在Gemini answers anything.

Disk Reallocate

我是1T的盘，本来的划分方案是C盘150G，D盘757G。这显然不太合理，C盘总红，D盘拥挤。

重新划完大概是C盘250G，D盘300G，E、F盘各200G这样。用windows自己的磁盘管理重分配就行。

磁盘备份和恢复镜像采用Hasleo Backup Suite，好用的。

Environment

这一世我要当环境管理糕手

Language

Python

我要用pyenv了，conda桑，沙扬娜拉。

https://github.com/pyenv/pyenv

  • pyenv-win默认且只能装到C盘，迁移方法：将.pyenv移到其他盘符下的新建目录，将PYENV三个环境变量改为指向这个新的.pyenv/pyenv-win，然后将Path里的bin和shims也改了
  • 在管理应用执行别名中关闭python.exe和python3.exe自动跳转微软商店的行为

使用

  • pyenv install x.xx.x安装特定版本
  • pyenv versions列出已安装版本
  • pyenv shell —— 仅当前 shell 会话有效
  • pyenv local —— 当前目录及子目录自动选择
  • pyenv global —— 全局（用户级）选择
